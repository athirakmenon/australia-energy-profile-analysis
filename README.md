# Australia Energy Profile Analysis

## Overview
Analysis of Australia's renewable energy consumption and 
carbon intensity trends from 1970 to 2022 across 6 continents.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, NumPy, PIL

## Key Findings
- Australia recorded highest carbon intensity globally
- Significant growth in hydroelectricity production from 1970-2022
- Heavy reliance on fossil fuels persists in 2022

## Data Source
Global energy consumption dataset
